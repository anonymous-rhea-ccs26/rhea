File Title: Turfgrass Carbon Meta-Analysis Main Code
File Name: TurfgrassCarbonMeta-analysis_v1.R
File Description: R script to compute whole-profile soil C sequestration rates and associated errors for each dataset, perform meta-analyses, and generate figures for the manuscript "High soil carbon sequestration rates persist several decades in turfgrass systems: a meta-analysis".

File Title: ModifiedVanHadenESMScript
File Name: ModifiedVanHadenESMScript.R
File Description: R script to compute soil C stocks by equivalent soil mass method. Modified from code provided by Von Haden et al., 2020 (DOI: 10.1111/gcb.15124).

File Title: CarbonDatabase
File Name: CarbonDatabase_Submit_Oct2022.xlsx
File Description: Contains descriptions of each study, summary values for sequestration rates by sampling interval and whole depth, and computations for emissions from maintenance activities.

++++++

File Title: All Turfgrass Data_By Layer
File Name: All_Turfgrass_Data_By Layer.xlsx
File Description: Soil carbon concentration, bulk density, and carbon stock measured in individual soil layers. These data were used to re-compute stocks by equivalent soil mass in contrast to fixed depth accounting. These data are duplicates of data provided in individual files for each study. Not all studies reported depth-discrete data, so not all studies are included here.
